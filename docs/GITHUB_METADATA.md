## 설정 위치

GitHub 저장소 페이지 오른쪽 About 영역에서 톱니바퀴 또는 Edit repository details를 눌러 설정한다.

브라우저가 GitHub에 로그인되어 있지 않거나 GitHub CLI가 설치되어 있지 않으면 자동 설정이 불가능하다.
